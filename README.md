# Cosmic filaments

This repository contains my work on building a cosmic filament catalog suitable for stacking in X-rays with eROSITA. 

This work was conducted at Max Planck for Extraterrestrial Physics between November 2022 and Jan 2023. For a summary of my work, please refer to the `Project_presentation.pdf`.

Also made available is the `data.tar.gz` file generated as part of this project [here](https://drive.google.com/file/d/1UDlMP1ElFK5CghM4_yxwQVJzyL7edbBR/view?usp=sharing). These files contain the [Cosmic filament catalogue](https://l3s.osups.universite-paris-saclay.fr/cosfil.html) from the [ByoPic](https://byopic.eu/) team, as described in Malavasi et al. 2020, A&A, 642, A19, which are reformatted using my scripts (see `python_files/reformatting_Malavasi_catalogue2fits.py`) into `fits` files for convenient use.
