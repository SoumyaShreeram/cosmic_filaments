# Cosmic filaments

This repository contains my work on building a cosmic-filament catalog suitable for stacking in X-rays with eROSITA. 

This work was conducted at Max Planck for Extraterrestrial Physics between Nov 2022 - Jan 2023. For summary of my work please refer to the `Project_presentation.pdf`.

Also made available is the `data.tar.gz` file generated as part of this project: https://drive.google.com/file/d/1UDlMP1ElFK5CghM4_yxwQVJzyL7edbBR/view?usp=sharing

This contains the [Cosmic filament catalogue](https://l3s.osups.universite-paris-saclay.fr/cosfil.html) from the [ByoPic](https://byopic.eu/) team and described in Malavasi et al. 2020, A&A, 642, A19 reformatted into `fits` files, for convinient useage.
